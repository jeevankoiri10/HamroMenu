# Hamro menu
Hamro Menu is a Flutter App developed by Jeevan Koiri.

## Descriptions

Our app is capable of finding the nearby restaurant's menu. We can search for nearby restaurants and it will display the list of restaurants of the area pointed out in map, then when click on specific hotel, displays the details of that hotel along with its photoes and reviews. This was the 6th sem Minor project.
